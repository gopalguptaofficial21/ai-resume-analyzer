# AI Resume Analyzer

An AI-powered resume analyzer built with Python and Streamlit that automatically extracts skills from PDF resumes and matches them with job roles.

## Features
- Upload any resume in PDF format
- Automatically extracts AI and technical skills
- Shows job role match percentage for AI/ML Trainer, Data Analyst, ML Engineer, Data Scientist
- Displays missing skills to improve
- Overall AI skills score with recommendations

## Technologies Used
- Python
- Streamlit
- PyPDF2
- Natural Language Processing (NLP)
- Scikit-learn

## How to Run

Install dependencies:
pip install streamlit pandas PyPDF2 scikit-learn

Run the app:
streamlit run app.py
